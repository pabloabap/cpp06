#Casting
##From_c_type_conversion
Existe el casteo como en C, poniendo el tipo de dato entre parentesis,
el casteo implicito, cuando se puede realizar la conversion directamente
al haber tipos de datos similares (int to doubel)
```c
void	dump_32bits_integer( int const n );
void	dump_64bits_double( double const z );


int  main( void )
{
	int	a = 42;
	double	b = a;		//Implicit conversion cast
	double	c = (double) a; //Explicit conversion cast
	
	double	d = a;		// Implicit promotion
	int	e = d;		// Implicit demotion. Harzrdous
	int	f = (int) d;	// Explicit demotion -> OK you are in change

	dump_32bits_integer( a );

	dump_64bits_double( b );
	dump_64bits_double( c );

	dump_64bits_double( d );
	dump_32bits_integer( e );
	dump_32bits_integer( f );
	
	return ( 0 );
}
```
